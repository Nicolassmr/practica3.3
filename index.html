Navegadores Web

Un navegador web (en ingles, web browser) es un software, aplicacion o programa que permite el acceso a la Web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad badica de un navegador web es permitir la visualizacino de documentos de texto, posiblemente con recursos multimedia incrustados

Mozillla Firefox es n navegador web libre y codigo abierto desarrilado para Linuz, Android, iOS, macOS y Microsoft Windows coordinado por la Corporacion Mozilla y la Fundacion Mozila

Google Chorme: Google Chrome es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de protectos de codigo abierto
